# Character Info

Contains all information about a character.

- **ID**: 0x09
- **Size**: Variable
- **LZ4 Compressed**: Yes
- **State**: Joining
- **Bound To**: Client <-> Server

## Structure

```
//------------------------------------------------
//--- 010 Editor v12.0 Binary Template
//
//      File: SM_Packet_9
//   Authors: ColdMeekly
//   Version: 1.0.0
//   Purpose: Yes
//  Category: 
// File Mask: 
//  ID Bytes: 09 FA 7F
//   History: 
//------------------------------------------------

typedef struct {
    byte data[16];
} uuid;

typedef struct {
  
  byte id;
  char magic[2];
  byte Hair; // A3 = leftmost hair, A7 = rightmost hair
  byte name_len;
  char name[name_len]; // TODO: Check how name affects packet
  char magic_char[4]; // Check if magic is 00 00 00 02
  byte is_male;
  byte num_uuids;
  uuid uuids[num_uuids];
  BigEndian();
  uint32 skin_id; // 0-4
  LittleEndian();
  char magic2[11];
} SMPacket9 <optimize=false>;


FSeek(0);
```

|------------|------------ |------|-------|
| ID | u8 | 1 byte |
| Magic | u8[2] | 2 bytes |
| Hair | u8 | 1 byte | 0xA3 = leftmost hair, 0xA7 = rightmost hair |
| Name Length | u8 | 1 byte |
| Name | char[Name Length] | Name Length bytes |
| Magic 2 | u16 | 2 bytes | |
| Is Male | bool | 1 byte | |
| UUID Count | u8 | 1 byte | |
| UUIDs | uuid[] | 16 x UUID Count bytes | |
| Skin ID | u32 | 4 bytes | 0-4 |
| Magic 3 | u8[11] | 11 bytes | |
