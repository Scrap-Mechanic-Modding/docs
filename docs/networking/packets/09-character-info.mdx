# Character Info

Contains all information about a character.

- **ID**: 0x09
- **Size**: Variable
- **LZ4 Compressed**: Yes
- **State**: Joining
- **Bound To**: Client -> Server

## Structure

| Field Name | Field Type  | Size | Notes |
|------------|------------ |------|-------|
| Name Legth | UInt16 | 2 | |
| Name | String | Name Legth | |
| CharVersionMaybe | UInt32 | 4 | Always 2? |
| IsFemale | Boolean | 1 | |
| Clothes Length | Byte | 1 | |
| Clothes | UUID | 16 * Items Length | Uuid fields from `Data\Character\customization_options.json`, 1 from each category (Face, Hair, Torso..) |
| FacePalette | UInt32 | 4 | Index into palette inside Face of `Data\Character\customization_options.json` |
| HairPalette | UInt32 | 4 | Index into palette inside Hair of `Data\Character\customization_options.json` |
| FacialHairPalette | UInt32 | 4 | Unused for now |
| TorsoPalette | UInt32 | 4 | Unused for now |
| GlovesPalette | UInt32 | 4 | Unused for now |
| ShoesPalette | UInt32 | 4 | Unused for now |
| PantsPalette | UInt32 | 4 | Unused for now |
| HatPalette | UInt32 | 4 | Unused for now |
| BackpackPalette | UInt32 | 4 | Unused for now |
