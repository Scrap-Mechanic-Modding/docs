# Glowstick Effect Removal

This packet is sent when the player removes a glowstick effect.

- **ID**: 0x28 (40)
- **Size**: 6
- **LZ4 Compressed**: No
- **State**: Playing
- **Bound To**: Client IDK Server

## Structure

| Offset | Field Name       | Field Type | Notes |
|--------|------------------|------------|-------|
| 0x00   | Unknown          | u8         | Unknown |
| 0x01   | Unknown          | u8         | Unknown |
| 0x02   | Effect ID        | u32        |  |
