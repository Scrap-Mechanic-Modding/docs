# Lift Delete Creation

This packet is sent when the player deletes a creation of their lift.

- **ID**: 0x2B (43)
- **Size**: 5
- **LZ4 Compressed**: No
- **State**: Playing
- **Bound To**: Client -> Server

## Structure

| Offset | Field Name       | Field Type | Notes |
|--------|------------------|------------|-------|
| 0x01   | Unknown          | u8         | 40 |
| 0x02   | Lift ID          | u32        |  |
