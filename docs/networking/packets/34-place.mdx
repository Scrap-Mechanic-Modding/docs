# Place

This packet is sent when the player places a shape.

- **ID**: 0x22 (34)
- **Size**: Variable
- **LZ4 Compressed**: No
- **State**: Playing
- **Bound To**: Client -> Server

## Structure

Depends on UUID and drag size.

#### Place 1x1x1 block
| Offset | Field Name       | Field Type | Notes |
|--------|------------------|------------|-------|
| 0x00   | Unknown          | u8         | |
| 0x04   | HotbarIndex      | u8         | |
| 0x25   | ShapeUuid        | be u128    | |
| 0x38   | Pos1             | float      | Unknown axis |
| 0x3C   | Pos2             | float      | Unknown axis |
